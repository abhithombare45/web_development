Project 1: Bangalore House Price Prediction

This project is an end-to-end machine learning application that predicts house prices in Bangalore based on features such as location, size, and amenities.

It is built using Python, Pandas, Scikit-learn for model training and deployed with Flask, Gunicorn, and NGINX on AWS EC2.

Key highlights include:

  • Data preprocessing & feature engineering
  • Hyperparameter tuning for regression models
  • Interactive web interface with Flask
  • Full-stack deployment pipeline (EC2 + NGINX + Gunicorn)
